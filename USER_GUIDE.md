# 📖 دليل استخدام التطبيق - User Guide
## نظام إدارة المدرسة الابتدائية

---

## 🎯 نظرة عامة

هذا النظام يساعد في إدارة المدرسة الابتدائية بشكل رقمي، يتضمن:
- إدارة التلاميذ والأقسام
- إدارة المواد الدراسية
- تسجيل النتائج (اختبارات وامتحانات)
- حساب المعدلات والإحصائيات
- طباعة كشوف الدرجات
- تصدير/استيراد البيانات

---

## 🚀 بدء الاستخدام

### 1. تسجيل الدخول

#### الخطوة الأولى: فتح التطبيق
- افتح المتصفح واذهب إلى: `http://localhost:3001` (أو العنوان المحدد)

#### الخطوة الثانية: إدخال البيانات
ستظهر لك صفحة تسجيل الدخول:

```
┌─────────────────────────────┐
│    📚 تسجيل الدخول           │
│  نظام إدارة المدرسة          │
│                             │
│  اسم المستخدم: [admin]      │
│  كلمة المرور: [••••••]      │
│                             │
│      [دخول]                 │
└─────────────────────────────┘
```

- **اسم المستخدم**: أدخل اسم المستخدم (مثال: `admin`)
- **كلمة المرور**: أدخل كلمة المرور

#### ما يحدث في الخلفية:
1. التطبيق يرسل البيانات إلى الخادم (`/api/token/`)
2. الخادم يتحقق من البيانات
3. إذا كانت صحيحة، يُرجع **JWT Token** (رمز الأمان)
4. التطبيق يحفظ التوكن في المتصفح
5. يتم توجيهك إلى **لوحة التحكم**

## 🏠 لوحة التحكم (Dashboard)

بعد تسجيل الدخول، سترى الصفحة الرئيسية:

### البطاقات الإحصائية:
```
┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐
│ عدد      │  │ عدد      │  │ نسبة     │  │ آخر      │
│ التلاميذ │  │ الأقسام  │  │ النجاح   │  │ النتائج  │
│   150    │  │    12    │  │   85%    │  │   450    │
└──────────┘  └──────────┘  └──────────┘  └──────────┘
```

- تعرض هذه البطاقات إحصائيات سريعة
- يتم تحديثها تلقائياً عند تغيير البيانات

### الروابط السريعة:
- **إضافة تلميذ جديد** → ينقلك إلى صفحة الطلاب
- **إدخال النتائج** → ينقلك إلى صفحة النتائج
- **طباعة الكشوفات** → ينقلك إلى صفحة النتائج (وضع الطباعة)

## 👥 إدارة الطلاب

### الوصول إلى الصفحة:
من القائمة الجانبية → **التلاميذ** أو من Dashboard → **إضافة تلميذ جديد**

### 1. إضافة طالب جديد

#### الخطوات:
1. اضغط على زر **"+ إضافة تلميذ"** (في الأعلى)
2. ستظهر نافذة منبثقة (Modal) تحتوي على نموذج:

```
┌──────────────────────────────────────┐
│  [X] إضافة تلميذ جديد                 │
├──────────────────────────────────────┤
│  الاسم الكامل: [_______________]      │
│  تاريخ الميلاد: [__/__/____]         │
│  الجنس: ○ ذكر  ○ أنثى                │
│  القسم: [اختر القسم ▼]                │
│  رقم ولي الأمر: [_______________]     │
│  العنوان: [_______________]           │
│  ملاحظات: [_______________]           │
│                                       │
│        [حفظ]      [إلغاء]             │
└──────────────────────────────────────┘
```

3. املأ البيانات المطلوبة:
   - **الاسم الكامل**: إلزامي ⚠️
   - **تاريخ الميلاد**: إلزامي ⚠️
   - **الجنس**: اختار ذكر أو أنثى
   - **القسم**: اختار القسم من القائمة
   - **رقم ولي الأمر**: اختياري
   - **العنوان**: اختياري
   - **ملاحظات**: اختياري

4. اضغط **"حفظ"**

#### ما يحدث:
- التطبيق يرسل البيانات إلى الخادم (`POST /api/students/`)
- الخادم يحفظ الطالب في قاعدة البيانات
- التطبيق يعرض إشعار نجاح (✓)
- يتم تحديث الجدول تلقائياً

---

### 2. عرض قائمة الطلاب

#### الجدول يعرض:
```
┌──────┬──────────────┬──────────┬───────┬────────────┐
│ رقم  │ الاسم        │ القسم   │ المعدل│ الإجراءات │
├──────┼──────────────┼──────────┼───────┼────────────┤
│ 1    │ أحمد محمد    │ قسم 1 أ │ 15.50 │ [عرض] [تعديل] [حذف] │
│ 2    │ فاطمة علي    │ قسم 1 ب │ 16.20 │ [عرض] [تعديل] [حذف] │
└──────┴──────────────┴──────────┴───────┴────────────┘
```

#### البحث والتصفية:
- **البحث**: اكتب في مربع البحث لفلترة الطلاب حسب الاسم
- **تصفية حسب القسم**: اختر القسم من القائمة المنسدلة

---

### 3. تعديل طالب

#### الخطوات:
1. في الجدول، اضغط على زر **"تعديل"** (أيقونة القلم)
2. ستفتح نفس النافذة المنبثقة مع بيانات الطالب الحالية
3. عدّل البيانات التي تريد تغييرها
4. اضغط **"حفظ"**

---

### 4. حذف طالب

#### الخطوات:
1. في الجدول، اضغط على زر **"حذف"** (أيقونة سلة المهملات)
2. ستظهر رسالة تأكيد: "هل أنت متأكد من حذف هذا التلميذ؟"
3. اضغط **"موافق"** للتأكيد

⚠️ **تحذير**: سيتم حذف جميع النتائج المرتبطة بهذا الطالب تلقائياً!

---

### 5. عرض كشف الدرجات (Report Card)

#### الخطوات:
1. في الجدول، اضغط على زر **"عرض"** (أيقونة الملف)
2. ستفتح نافذة تعرض:
   - معلومات الطالب
   - قائمة المواد الدراسية
   - الدرجات لكل مادة (اختبار + امتحان + المعدل)
   - المعدل العام للفصل
   - حالة النجاح/الرسوب

#### اختيار الفصل الدراسي:
- استخدم القائمة المنسدلة لتغيير الفصل (1, 2, 3)

---

### 6. تصدير قائمة الطلاب

#### الخطوات:
1. اضغط على زر **"تصدير CSV"** (في الأعلى)
2. سيتم تنزيل ملف Excel يحتوي على:
   - الاسم الكامل
   - تاريخ الميلاد
   - الجنس
   - القسم
   - رقم ولي الأمر
   - العنوان

## 📚 إدارة الأقسام والمواد

### إدارة الأقسام (Classes):
1. من القائمة الجانبية → **الأقسام**
2. **إضافة قسم**: اضغط "+ إضافة قسم"
   - أدخل اسم القسم (مثال: "قسم 1 أ")
   - حدد المستوى (1-6)
3. **تعديل/حذف**: نفس طريقة الطلاب

### إدارة المواد (Subjects):
1. من القائمة الجانبية → **المواد**
2. **إضافة مادة**: اضغط "+ إضافة مادة"
   - أدخل اسم المادة (مثال: "الرياضيات")
   - حدد المعامل (coefficient) - يستخدم لحساب المعدل الموزون
   - حدد المستوى (1-6)
3. **تعديل/حذف**: نفس طريقة الطلاب

---

## 📝 إدارة النتائج

### الوصول:
من القائمة الجانبية → **النتائج**

### الصفحة تحتوي على:

#### 1. الفلاتر (في الأعلى):
```
القسم: [اختر القسم ▼]
الفصل الدراسي: ○ 1  ○ 2  ○ 3
نوع التقييم: ○ اختبار  ○ امتحان
وضع العرض: ○ مادة واحدة  ○ جميع المواد
```

#### 2. وضع العرض: "مادة واحدة" (Single Mode)

##### الخطوات لإدخال النتائج:
1. اختر **القسم** من القائمة
2. اختر **الفصل الدراسي** (1, 2, 3)
3. اختر **نوع التقييم** (اختبار أو امتحان)
4. اختر **المادة** من القائمة
5. ستظهر قائمة بجميع الطلاب في القسم:

```
┌────────────────┬──────────┐
│ اسم الطالب     │ النقطة   │
├────────────────┼──────────┤
│ أحمد محمد      │ [15.5]   │
│ فاطمة علي      │ [16.0]   │
│ خالد حسن       │ [14.5]   │
└────────────────┴──────────┘
```

6. أدخل الدرجات لكل طالب (0-20)
7. اضغط **"حفظ"**

##### ما يحدث:
- التطبيق يحفظ الدرجات في قاعدة البيانات
- لكل طالب/مادة/فصل/نوع، يمكن حفظ **نقطة واحدة فقط**
- إذا أدخلت نقطة جديدة، سيتم استبدال القديمة

---

#### 3. وضع العرض: "جميع المواد" (Grid Mode)

##### الخطوات:
1. اختر **القسم** و **الفصل** و **نوع التقييم**
2. اضغط على زر **"جميع المواد"** (Grid Mode)
3. ستظهر جدول كبير:

```
┌──────────┬──────┬──────┬──────┬──────┐
│ الطالب   │ ريا. │ عربي │ فرنسي│ علوم │
├──────────┼──────┼──────┼──────┼──────┤
│ أحمد     │ 15.5 │ 16.0 │ 14.5 │ 15.0 │
│ فاطمة    │ 16.5 │ 17.0 │ 15.5 │ 16.0 │
└──────────┴──────┴──────┴──────┴──────┘
```

4. أدخل جميع الدرجات في نفس الوقت
5. اضغط **"حفظ"**

---

#### 4. حساب المعدل

##### كيف يعمل:
- التطبيق يحسب المعدل تلقائياً لكل طالب
- **إذا وُجد اختبار وامتحان**: المعدل = `(اختبار + امتحان) / 2`
- **إذا وُجد واحد فقط**: يستخدم القيمة المتوفرة
- **المعدل الموزون**: `Σ(درجة × معامل) / Σ(معامل)`

##### مثال:
- مادة الرياضيات: اختبار 15 + امتحان 18 = **المعدل: 16.5**
- مادة العربية: اختبار 16 + امتحان 16 = **المعدل: 16.0**
- المعاملات: رياضيات (3) + عربية (2)
- **المعدل العام**: `(16.5×3 + 16.0×2) / (3+2) = 16.3`

---

#### 5. طباعة كشف الدرجات

##### الخطوات:
1. اختر القسم والفصل
2. اضغط على زر **"طباعة"** (Printer icon)
3. سيتم فتح نافذة الطباعة:

```
┌────────────────────────────────────────┐
│  مدرسة الابتدائية                       │
│  كشف درجات - الفصل 1                    │
│  قسم: 1 أ                               │
│                                        │
│  ┌────────┬──────┬──────┬──────┬──────┐│
│  │ الطالب │ رياضيات│ عربي │ معدل│ │
│  ├────────┼──────┼──────┼──────┼──────┤│
│  │ أحمد   │ 16.5 │ 16.0 │ 16.3 │ ناجح││
│  └────────┴──────┴──────┴──────┴──────┘│
│                                        │
│  [Ctrl+P للطباعة]                      │
└────────────────────────────────────────┘
```

4. اضغط `Ctrl+P` (Windows) أو `Cmd+P` (Mac) لطباعة

---

#### 6. تصدير قالب Excel

##### الخطوات:
1. اختر القسم والفصل ونوع التقييم
2. اختر وضع العرض (مادة واحدة أو جميع المواد)
3. اضغط على زر **"تحميل القالب"** (Download Template)
4. سيتم تنزيل ملف Excel يحتوي على:
   - أسماء الطلاب
   - أعمدة للمواد (أو عمود واحد إذا كان وضع "مادة واحدة")
   - بيانات فارغة لإدخال الدرجات

##### استخدام القالب:
1. افتح الملف في Excel
2. أدخل الدرجات
3. احفظ الملف
4. في التطبيق، اضغط **"رفع ملف Excel"** (Upload)
5. اختر الملف
6. اضغط **"استيراد"**

من القائمة الجانبية → **الإحصائيات**

## 🔐 الأمان والحماية

### المصادقة (Authentication):
- جميع الصفحات (عدا صفحة تسجيل الدخول) تتطلب تسجيل دخول
- إذا انتهت صلاحية التوكن، سيتم تسجيل خروجك تلقائياً

### حفظ البيانات:
- جميع البيانات محفوظة في قاعدة البيانات
- يتم إرسال الطلبات عبر HTTPS (في الإنتاج)

## 💡 نصائح للاستخدام

### 1. إضافة البيانات بالترتيب:
1. أولاً: **الأقسام** (لأن الطلاب يحتاجون إلى أقسام)
2. ثانياً: **المواد** (لأن النتائج تحتاج إلى مواد)
3. ثالثاً: **الطلاب** (يربطون بالأقسام)
4. رابعاً: **النتائج** (تربط الطلاب بالمواد)

### 2. استخدم وضع "جميع المواد" لتوفير الوقت:
- أسرع لإدخال جميع الدرجات دفعة واحدة
- مناسب عند إدخال نتائج الفصل بأكمله

### 3. احفظ بانتظام:
- بعد إدخال الدرجات، اضغط "حفظ" فوراً
- تجنب فقدان البيانات

### 4. استخدم Excel للإدخال الجماعي:
- مناسب عند إدخال نتائج عدة طلاب في نفس الوقت
- يمكن لعدة أشخاص العمل على نفس الملف

## ⚠️ أخطاء شائعة وحلولها

### 1. "البريد الإلكتروني أو كلمة المرور غير صحيحة"
- **السبب**: بيانات الدخول خاطئة
- **الحل**: تحقق من اسم المستخدم وكلمة المرور

### 2. "لا يمكن الوصول إلى الخادم"
- **السبب**: الخادم (Backend) غير شغال
- **الحل**: 
  - تأكد من تشغيل Backend (`python manage.py runserver`)
  - تحقق من عنوان URL في إعدادات Frontend

### 3. "القسم فارغ"
- **السبب**: لم تختر قسماً أو القسم لا يحتوي على طلاب
- **الحل**: اختر قسماً يحتوي على طلاب

### 4. "المعدل لا يظهر"
- **السبب**: لا توجد نتائج للطالب في الفصل المختار
- **الحل**: تأكد من إدخال نتائج للطالب في هذا الفصل

### 5. "خطأ في رفع الملف"
- **السبب**: تنسيق الملف غير صحيح أو الأعمدة غير متطابقة
- **الحل**: استخدم القالب الذي يُولده التطبيق (`تحميل القالب`)

```

   ↓

```
